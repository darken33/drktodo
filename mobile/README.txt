README
------------------------------------------------------------------------

drkTodo est une simple application représentant une liste de chose à 
faire pour téléphone android. Cette application est une application
hybride développée grace au technologies html5, jquerymobile et
phonegap (cordova).

Elle vous permet de lister simplement vos différentes choses à faire. En
outre elle mémorise les 15 dernières taches entrées.

A noter que ce logiciel est un logiciel libre et qu'il respecte les
quatres principes fondamentaux suivants :
 * Liberté 0 : La liberté d’exécuter le programme, pour tous les usages.
 * Liberté 1 : La liberté d’étudier le fonctionnement du programme.
 * Liberté 2 : La liberté de redistribuer des copies.
 * Liberté 3 : La liberté d’améliorer le programme et de publier ses 
               améliorations.
